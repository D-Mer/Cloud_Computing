# 文件内容说明

这里只留下了之前保留的部分图数据，全局数据离散点很多，效果不好。以weight-n.gexf预览效果最佳，是去除离群点(绝大部分是无效数据)后的全局图。.gexf是图文件，用gephi等软件可直接打开，如何展示可diy，推荐echarts
